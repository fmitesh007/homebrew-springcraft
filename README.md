# homebrew-springcraft

Homebrew tap for [springcraft](https://github.com/fmitesh007/springcraft-cli)

## Installation

```bash
# Add the tap
brew tap fmitesh007/springcraft

# Install springcraft
brew install springcraft
```

## Usage

```bash
springcraft .
```

## Uninstall

```bash
brew uninstall springcraft
brew untap fmitesh007/springcraft
```
